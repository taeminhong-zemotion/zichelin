# 고쿠텐

메뉴: 고쿠텐동, 새우텐동

위치: 카카오맵 (opens new window)

주변에서 보기드물게 튀김 덮밥을 파는 곳입니다. 튀김 하나는 정말 손색이 없습니다. 가게가 작아서 항상 대기줄이 길게 생기기 때문에 일찍 가야합니다.

# 바이투헤븐

메뉴: 버거

위치: 카카오맵 (opens new window)

어설픈 수제 햄버거 가게에서 짜증이 날 때가 한두 번이 아녀서 처음 주문할 때 걱정했는데 그런 제 우려를 박살 내버렸습니다. 가격이 조금 낮았다면 더 좋았겠지만, 강남이니 그 정도는 고려해야겠죠. 햄버거를 싫어하는 것이 아니라면 강력히 추천합니다.

# 왓쇼이켄 라멘

메뉴: 돈코츠라멘, 미소라멘

위치: 카카오맵 (opens new window)

차슈가 상당히 푸짐하며 국물과 면도 수준급입니다. 근처에 몇 군데 라멘 가게가 더 있지만 여기가 한 끗 더 낫더라고요.

# 남도애찬

메뉴: 갈비탕, 영양돌솥밥, 매운갈비찜

위치: 카카오맵 (opens new window)

밑반찬이 굉장히 맛있어요. 점심 메뉴 하나하가 매우 훌륭했습니다.

# 농민백암왕순대

메뉴: 순대국밥

위치: 카카오맵 (opens new window)

강남에서 이 정도 가격에 이렇게 푸짐하고 맛있는 순대국밥을 먹을 수 있는 곳은 드물죠. 다만 대기줄이 워낙 길어서 먹기가 어렵습니다.

# 카츠하나

메뉴: 치킨카츠, 히레카츠

위치: 카카오맵 (opens new window)

닭다리 살로 만들어서 퍽퍽하지 않고 촉촉한 치킨카츠가 일품입니다. 히레카츠도 비계가 두툼하게 붙어있어 다른 가계의 돈가스와 상당히 다른 맛을 냅니다. 대체로 닭과 돼지의 지방이 풍부해서 고소하고 육향이 느껴지는 것이 특징입니다.

# 초선과여포

메뉴: 짜장면, 짬뽕, 삼선볶음밥

위치: 카카오맵 (opens new window)

주방장이 굉장히 실력이 좋습니다. 위치가 강남이라는 점을 고려한다면 가격도 만족스럽고요. 볶음밥 같은 기본 메뉴도 훌륭하지만 여기서 꿔바로우를 먹고 반해버렸습니다.

# 걸신맛집

메뉴: 강된장보리밥, 묵은지김치찜, 새싹비빔밥

위치: 카카오맵 (opens new window)

남도애찬보다 좀 더 구수하고 토속적인 전라도 음식을 제공합니다. 밥을 시킬 때 솥 밥 먹겠다고 따로 이야기하지 않으면 그냥 공기밥이 나오니 주문할 때 주의하세요.

# 생활맥주

메뉴: 앵그리버드(치킨), 미미네생활떡볶이

위치: 카카오맵 (opens new window)

처음에 맥주 맛에 놀라고 치킨을 먹고 두 번째 놀라고, 마지막으로 떡볶이를 먹고 또 놀랐습니다. 이곳에서 점심을 먹을 수는 없지만, 충분히 언급할 가치가 있는 곳입니다.
