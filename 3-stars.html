# 고쿠텐

메뉴: 고쿠텐동, 새우텐동

위치: 카카오맵 (opens new window)

주변에서 보기드문 튀김 덮밥을 먹을 수 있는 곳입니다. 튀김만 보자면 다른 곳을 압도합니다.

# 왓쇼이켄 라멘

메뉴: 돈코츠라멘, 미소라멘

위치: 카카오맵 (opens new window)

사장님이 부담스러울 정도로 친절합니다. 차슈가 상당히 푸짐하며 국물과 면도 수준급입니다.

# 남도애찬

메뉴: 갈비탕, 영양돌솥밥, 매운갈비찜

위치: 카카오맵 (opens new window)

밑반찬이 굉장히 맛있어요. 점심메뉴 하나하가 매우 훌륭했습니다.

# 농민백암왕순대

메뉴: 순대국밥

위치: 카카오맵 (opens new window)

강남에서 이 정도 가격에 이렇게 푸짐하고 맛있는 순대국밥을 먹을 수 있는 곳은 드물죠.

# 카츠하나

메뉴: 치킨카츠, 히레카츠

위치: 카카오맵 (opens new window)

닭다리 살로 만들어서 퍽퍽하지 않고 촉촉한 치킨카츠가 일품입니다. 히레카츠도 비계가 두툼하게 붙어있어 다른 가계의 돈가스와 상당히 다른 맛을 냅니다. 대체로 닭과 돼지의 지방이 풍부해서 고소하고 육향이 느껴지는 것이 특징입니다.

# 초선과여포

메뉴: 짜장면, 짬뽕, 삼선볶음밥

위치: 카카오맵 (opens new window)

주방장이 굉장히 실력이 좋습니다. 가격도 강남임을 고려한다면 저렴한 편이고요. 볶음밥 같은 기본 메뉴도 훌륭하지만 여기서 꿔바로우를 먹고 반해버렸습니다.
