# 짬뽕지존

메뉴: 수제비짬뽕, 쌀국수짬뽕, 순두부짬뽕

위치: 카카오맵 (opens new window)

중화요리 중에서 짬뽕이 특시 땡긴다면 이만한 곳이 없죠. 수제비짬뽕과 순두부짬뽕이 굉장히 맛있습니다. 짬뽕 하나에 1만원이 넘어간다는 점을 빼면 굉장히 만족스럽습니다.

# 재래미 2.0

메뉴: 탄탄면, 우육탕면, 딤섬

위치: 카카오맵 (opens new window)

대만/홍콩 음식전문점입니다. 저렴하게 딤섬을 즐길 수 있고 우육탕면과 탄탄면이 괜찮습니다. 탄탄면공방보다 조금 대만 현지에 가까운 맛을 즐기려면 이쪽을 추천합니다.

# 탄탄면공방

메뉴: 탄탄면, 탄탄국밥

위치: 카카오맵 (opens new window)

가게 설명에도 쓰여 있듯이 현지 탄탄면과는 조금 다른 맛을 지향합니다. 일본식 라면과 탄탄면의 중간쯤 될까요? 진짜 탄탄면과 다르다고 하지만 다 떠나서 그냥 맛있습니다. 볶음 김치가 참 좋더라고요.

# 엄지칼국수

메뉴: 손칼국수, 들깨칼국수, 팥칼국수

위치: 카카오맵 (opens new window)

해산물이 아니라 닭 육수로 국물을 내는 것 같아요. 그래서 시원한 맛은 없지만 대신 진하고 묵직합니다. 들깨 칼국수는 꼭 한 번 드셔 보세요.

# 이태리부대찌개

메뉴: 부대찌개, 부대전골

위치: 카카오맵 (opens new window)

회사 현관에서 퍼져오는 음식 냄새의 주인공이죠. 제가 먹었던 다른 프랜차이즈 부대찌개보다 여기가 훨씬 푸짐하고 맛있었어요. 그만큼 찾는 사람도 많아서 자리가 잘 안나는 편입니다.

# 청담돈가스

메뉴: 로스가스, 히레가스

위치: 카카오맵 (opens new window)

합리적인 가격에 제법 준수한 돈가스를 맛볼 수 있습니다. 감탄사가 나올 정도는 아니지만, 우리가 돈가스에 기대하는 맛은 뽑아줍니다.

# 구스아일랜드

메뉴: 치킨오버라이스, 치지어메리칸스파게티, 클래식버거

위치: 카카오맵 (opens new window)

간혹 별로인 경우도 있지만 대체로 음식이 괜찮습니다. 무엇보다 시야가 넓게 트이고 흥겨운 분위기를 낼 수 있는 곳이 근처에 이곳을 제외하면 굉장히 드물거든요. 가게 분위기가 한 몫 하는 곳입니다.
